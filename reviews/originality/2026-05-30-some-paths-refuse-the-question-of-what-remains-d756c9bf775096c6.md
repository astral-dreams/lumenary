## Exact Claim

Before any continuity or residue analysis after self-negation, test whether the path stages self-negation as an event with an after. If a path refuses seeking, before-and-after, or self-owned attainment, the question of what remains may be an analyst artifact rather than a practitioner problem.

## Claim Units

- Some contemplative paths stage a search sequence, so they generate a legitimate downstream remainder question.
- Some paths refuse the search sequence itself, so remainder pressure is malformed there.
- A precondition test should precede continuity, custody, or residue coding.
- The modern harm clarified is achievement-contingent self-worth turned into compulsive auditing for emptiness, detachment, or egolessness.

Dogen's practice-realization unity blocks a simple cause-then-result reading, while Soto material also says Dogen emphasized active practice because realization is already present. The candidate's gate is textually plausible, but only if it codes the native practice grammar rather than broad tradition names.

## Unlike Statement

Unlike Huangbo, Dogen, Collingwood, Gadamer, VCE, and prior Lumenary no-management records, this idea's specific contribution is a precondition test before remainder coding: only ask what remains if the path first authorizes a self-negation event with an after.

## Anomaly Probe

Candidate: Rinzai koan and huatou practice deliberately uses an impossible or destabilizing question to create great doubt, while Dzogchen can deny path and result yet still require direct introduction, teacher relation, and integration.

Why it strains the model: If a path uses an apparently malformed question as medicine, then question-refusal is not enough to decide that remainder pressure is analyst-made. If no-path language still depends on transmission and repair holders, then the residue question may be relocated to practice ecology rather than abolished.

Confidence effect: Lowers confidence in a binary search-sequence versus search-refusal model. Raises confidence only for a more cautious codebook that includes search-refusal, question-as-medicine, no-distance-with-support, and undecidable.

## Falsifiable Predictions

- If: The precondition gate has independent coding value. Then: Blind coders using only doctrinal or instruction passages can sort cases into search-sequence, search-refusal, question-as-medicine, no-distance-with-support, and undecidable above chance before seeing warnings or repairs. Weakens if: Coders cannot agree, or their labels are just tradition labels, teacher style, or genre labels in disguise. Test surface: Inter-rater coding on SN 22.59, SN 22.95, Huangbo, Bendowa, Tannisho, one Dzogchen direct-introduction source, and one Rinzai koan source.
- If: Remainder pressure is not universal. Then: In search-sequence cases, warnings and repairs cluster around misidentification, clinging, effort, and post-release integration. In search-refusal cases, warnings cluster around seeking, monitoring, conceptualization, and self-grading. Weakens if: Warnings and failures in search-refusal cases are predicted just as well by support-holder loss, teacher absence, trauma, sleep loss, or institutional harm without using the precondition gate. Test surface: Split-source coding, code path grammar from one corpus and predict held-out warnings, repairs, and practitioner failure cases.
- If: The proposed practice serves only the named modern wound. Then: Stable over-auditors using Stop the Emptiness Audit report less self-grading and faster return to ordinary care than journaling-only or rest-only controls, without increased avoidance or reassurance seeking. Weakens if: The practice becomes a new checking ritual, increases avoidance of needed repair, worsens rumination, or performs no better than ordinary rest or social contact. Test surface: Two-week pilot with screened perfectionistic meditators and self-improvers, excluding OCD, scrupulosity, dissociation, severe depression, mania, psychosis, addiction withdrawal, trauma activation, and acute crisis.

Answers that reduce originality: Originality falls if practitioners say this is standard pedagogy under no-seeking, shusho itto, Other Power, direct introduction, koan great doubt, or ordinary spiritual-bypassing triage, and if the gate adds no decisions beyond existing teacher judgment.
